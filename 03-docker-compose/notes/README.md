# Docker Compose Notes

## What is Docker Compose?
Compose is a tool for defining and running multi-container Docker applications. With Compose, you use a YAML file to configure your application’s services. Then, with a single command, you create and start all the services from your configuration.

## Key benefits of using Docker Compose:
- **Simplified multi-container orchestration:** Easily manage applications with multiple services.
- **Single command setup:** `docker-compose up` builds, (re)creates, starts, and attaches to containers for a service.
- **Environment consistency:** Ensures your application runs the same in development, testing, and production.
- **Easy configuration:** Use a simple YAML file to define your application stack.
