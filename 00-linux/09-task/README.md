# Task 9: Process Management

## Objective
Learn how to manage running processes.

## Exercises
1. List all running processes on the system.
2. Open a long-running process in the background (e.g., `sleep 300 &`).
3. Find the process ID (PID) of the `sleep` process.
4. Kill the `sleep` process using its PID.
5. Verify that the `sleep` process is no longer running.
6. View a real-time list of running processes.
