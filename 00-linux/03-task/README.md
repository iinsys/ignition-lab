# Task 3: File Content Viewing

## Objective
Learn how to view the content of files in the terminal.

## Exercises
1. Create a file named `long-text.txt` with the following content:
   ```
   Line 1
   Line 2
   Line 3
   Line 4
   Line 5
   Line 6
   Line 7
   Line 8
   Line 9
   Line 10
   Line 11
   Line 12
   Line 13
   Line 14
   Line 15
   ```
2. View the entire content of the file at once.
3. View the content of the file page by page.
4. View the first 5 lines of the file.
5. View the last 5 lines of the file.
