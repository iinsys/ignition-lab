# Task 7: Archiving and Compression

## Objective
Learn how to create and extract archives.

## Exercises
1. Create a directory named `my-archive`.
2. Create three empty files inside `my-archive`: `file1.txt`, `file2.txt`, and `file3.txt`.
3. Create a `tar` archive of the `my-archive` directory named `my-archive.tar`.
4. Create a `tar.gz` archive of the `my-archive` directory named `my-archive.tar.gz`.
5. Delete the `my-archive` directory.
6. Extract the contents of `my-archive.tar.gz`.
7. Verify that the `my-archive` directory and its contents have been restored.
