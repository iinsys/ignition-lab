# Task 6: Searching for Text

## Objective
Learn how to search for text within files.

## Exercises
1. Create a file named `story.txt` with the following content:
   ```
   The quick brown fox jumps over the lazy dog.
   The dog was not amused.
   The fox, however, was very happy.
   ```
2. Search for the word "fox" in `story.txt`.
3. Search for the word "dog" in `story.txt`, ignoring case.
4. Search for all lines that do NOT contain the word "The".
5. Search for the word "was" and show the line numbers where it appears.
