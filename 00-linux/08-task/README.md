# Task 8: Users and Groups

## Objective
Learn how to manage users and groups.

## Exercises
(Note: These commands may require `sudo` privileges)
1. Create a new user named `testuser`.
2. Create a new group named `testgroup`.
3. Add `testuser` to the `testgroup`.
4. View the groups `testuser` belongs to.
5. Remove `testuser` from `testgroup`.
6. View the groups `testuser` belongs to again to confirm the change.
7. Delete the `testuser`.
8. Delete the `testgroup`.
