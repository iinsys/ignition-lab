# Task 5: Searching for Files

## Objective
Learn how to search for files and directories.

## Exercises
1. Find all files in your home directory with the `.log` extension.
2. Find all directories in `/var` named `log`.
3. Find all files in `/etc` that have been modified in the last 24 hours.
4. Find all files in your home directory that are larger than 1MB.
5. Find all empty files in your home directory.
