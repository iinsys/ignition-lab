# Task 1: File System Navigation

## Objective
Learn how to navigate the Linux file system.

## Exercises
1. Print the current working directory.
2. List the files and directories in the current directory.
3. List the files and directories in the `/` directory.
4. Change your current directory to `/etc`.
5. Print the current working directory to confirm you are in `/etc`.
6. Change your current directory back to your home directory using the `~` shortcut.
7. Print the current working directory to confirm you are back in your home directory.

## Solution
1. `pwd`
2. `ls`
3. `ls /`
4. `cd /etc`
5. `pwd`
6. `cd ~`
7. `pwd`
