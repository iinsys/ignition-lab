# Task 2: File and Directory Manipulation

## Objective
Learn how to create, delete, copy, and move files and directories.

## Exercises
1. Create a new directory called `test-dir`.
2. Create an empty file called `test-file.txt` inside `test-dir`.
3. Copy `test-file.txt` to a new file called `test-file-copy.txt` inside `test-dir`.
4. Rename `test-file.txt` to `test-file-renamed.txt`.
5. Move `test-file-copy.txt` to the parent directory (out of `test-dir`).
6. Delete `test-file-renamed.txt`.
7. Delete the `test-dir` directory.
