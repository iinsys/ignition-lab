# Task 4: File Permissions

## Objective
Learn how to manage file permissions.

## Exercises
1. Create a new file called `test.sh`.
2. View the current permissions of `test.sh`.
3. Make the file executable for the owner.
4. View the permissions again to confirm the change.
5. Make the file readable and writable for the group.
6. View the permissions again to confirm the change.
7. Remove all permissions for others.
8. View the permissions again to confirm the change.
9. Try to make the file writable for everyone.
10. View the permissions again to confirm the change.
