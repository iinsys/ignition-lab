# Networking Notes

## Basic Networking Concepts

- **IP Address:** A unique address that identifies a device on a network.
- **Subnet Mask:** A number that defines a range of IP addresses available to a network.
- **Gateway:** A node on a network that serves as an access point to another network.
- **DNS (Domain Name System):** A hierarchical and decentralized naming system for computers, services, or other resources connected to the Internet or a private network.
- **TCP (Transmission Control Protocol):** A connection-oriented protocol that provides reliable, ordered, and error-checked delivery of a stream of octets between applications running on hosts communicating over an IP network.
- **UDP (User Datagram Protocol):** A connectionless protocol that does not guarantee delivery, ordering, or duplicate protection.
- **Port:** An endpoint of communication in an operating system.
